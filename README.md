# 📋 Task Manager (Productivity System)

A structured and interactive task management web app designed to improve productivity using prioritization logic, real-time tracking, and intuitive UI.

---

## 🚀 Live Demo
👉 https://prakharojha960.github.io/task-manager/

---

## ✨ Features

### 📌 Core Functionality
- Add, edit, and delete tasks  
- Organize tasks using **priority-based system (Eisenhower Matrix)**  
- Drag-and-drop tasks across categories  
- Mark tasks as completed  

---

### ⏱ Time & Productivity Tracking
- Real-time countdown timers for deadlines  
- Automatic overdue detection  
- Task completion tracking with **time taken analysis**  
- Visual feedback for urgency and status  

---

### 🎯 UI/UX Highlights
- Clean and structured dashboard layout  
- Smooth drag-and-drop interaction  
- Custom dropdown menu with edit/delete actions  
- Instant UI updates (no reload required)  
- Minimal and intuitive design for fast usage  

---

## 🧠 Highlights

- Built completely from scratch (no frameworks)  
- Real-time DOM manipulation and updates  
- Custom task prioritization system (not a basic list)  
- Dynamic state management using LocalStorage  
- Designed with productivity-first approach  

---

## 🛠 Tech Stack

- **HTML** – Structure  
- **CSS** – Styling & layout  
- **JavaScript** – Logic & interactivity  
- **LocalStorage** – Data persistence  

---

## ⚙️ How It Works

1. Add tasks with optional deadlines  
2. Assign tasks to priority categories  
3. Drag tasks across sections based on importance/urgency  
4. Track time remaining using live countdown  
5. Mark tasks as completed and view completion time
